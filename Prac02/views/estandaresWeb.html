¿Qué es la W3C?

La W3C (World Wide Web Consortium) es una comunidad internacional que desarrolla estándares abiertos para asegurar el crecimiento de la Web.

Estándares destacados:

  • SVG: Gráficos vectoriales escalables para navegadores.
  • WOFF: Formato comprimido de fuentes para la web.
  • WebRTC: Comunicación en tiempo real entre navegadores (audio, video, datos).
  • XML: Lenguaje para almacenar y transportar datos.
